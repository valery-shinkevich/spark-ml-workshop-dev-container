# Dev-container for Spark ML Workshop

## Summary

*Dev-container for Spark ML Workshop. Includes JDK 11.0.13, Spark 3.1.2 (+ pyspark), Scala 2.12.10, Sbt 1.5.8, Python 3.6.15, pandas 1.3.5, scikit-learn 1.0.1*

Based on vscode-dev-contaniers sample.

## Using this definition with an existing folder

Note that only the integrated terminal is supported by the Remote - Containers extension. You may need to modify `launch.json` configurations to include the following value if an external console is used.

```json
"console": "integratedTerminal"
```
